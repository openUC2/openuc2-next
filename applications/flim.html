FLIM & Advanced Fluorescence

Fluorescence Lifetime Imaging Microscopy (FLIM) adds a new quantitative dimension to fluorescence data. openUC2 FRAME provides a FLIM-ready optical path — pair it with the FLIM Labs add-on module for phase-domain lifetime measurements on the same platform you already use for widefield and confocal-style imaging.

FRAME + FLIM Labs | 20× | 488 nm pulsed excitation

FLIM-ready from day one

FRAME's modular cube architecture lets you provision a FLIM upgrade path without rebuilding your microscope. The optical engine already provides clean spectral separation, fast channel switching, and stable excitation power — the prerequisites for quantitative lifetime imaging.

When you're ready to add FLIM, the FLIM Labs module slots into the existing cube stack, providing a calibrated pulsed-excitation source and time-correlated detector. A Docker-based software bridge connects FLIM Labs' acquisition software to ImSwitch, allowing simultaneous intensity and lifetime data collection.

Key capabilities

⏱️ Lifetime imaging

Measure fluorescence lifetimes in the nanosecond range. Distinguish spectrally overlapping fluorophores by their lifetime — not just their emission wavelength.

🔬 Galvo scanning

Galvo scanner cube enables fast point-scanning for confocal-style and FLIM acquisition. Compatible with FRAME's standard cube slot.

🎨 Quantitative flat-field

Built-in flat-field and shading correction; metadata-rich acquisition for reproducible inter-sample comparison of lifetime data.

🔧 Modular upgrade path

Start with widefield fluorescence today. Add the FLIM Labs module when your science demands lifetime data — no optical rebuild required.

📊 Phasor analysis

Phase-domain FLIM produces phasor plots that reveal molecular environment, FRET efficiency, metabolic state (NADH/FAD), and cell health without fitting.

🐳 Docker software bridge

FLIM Labs acquisition software runs in a Docker container alongside ImSwitch, with a REST bridge for synchronized image + lifetime data capture.
