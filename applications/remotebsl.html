Microscopy without entering the lab

In BSL-2 and BSL-3 laboratories, every entry is a contamination risk and a time cost. FRAME's fully remote browser-based operation means researchers can start, monitor, and stop experiments from outside the hot zone. The RESTful control interface integrates with existing lab automation and LIMS systems via standard APIs.

Hot-swap kinematic module mounts allow contaminated optical heads and filter sets to be swapped by technicians in the minimum donning time, with automatic re-alignment and module re-recognition by the software. Audit logs track who accessed the system, what acquisitions were made, and any hardware events — a requirement for BSL laboratory SOP compliance.

Key capabilities

🌐 Fully browser-based

Complete microscope control, live preview, acquisition scheduling, and data export from any browser. No specialist software to install on the remote machine.

🔒 Network hardened

VPN / Zero-Trust network overlay. Role-based access control separates viewers, operators, and administrators. All traffic encrypted in transit.

🔄 Hot-swap modules

Kinematic mounts with unique module IDs. Swap contaminated optical heads in seconds; ImSwitch auto-detects the new module and restores the last configuration.

📋 Audit log

Immutable, time-stamped log of all system events: logins, acquisitions, hardware faults, software updates. Ready for BSL SOP documentation.

🔬 Label-free & fluorescence

DPC / FPM for label-free phase imaging of unlabeled pathogens. Dual-color immunofluorescence for specific antigen detection on the same system.

🌡️ Incubator compatible

Same sealed, humidity-rated body as the organoid QC configuration. Run inside a BSL-certified CO₂ incubator with LAN feed-through.
