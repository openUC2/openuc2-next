How it works with openUC2

A standard fluorescence microscope consists of an excitation source, an excitation filter, a dichroic mirror, an emission filter, an objective, and a detector. In the openUC2 system, each of these components lives in its own cube module.

This means you can build exactly the fluorescence configuration you need: single-channel GFP, dual-color with 488/638 nm, or a full quad-color setup. Swap dichroics and filters by swapping cubes — no tools, no realignment.

Key capabilities

🎨 Multi-channel imaging

Up to 4–5 simultaneous fluorescence channels with filter-cube carousel. Fast switching minimizes photobleaching between channels.

💡 Low background

Baffled light paths and high-quality Chroma/Semrock filter sets minimize autofluorescence and cross-talk. Dark-field correction built into ImSwitch.

📐 Flat-field correction

Hardware-triggered flat-field calibration with reference slide. Homogeneous illumination across the full field of view for quantitative analysis.

🔬 High-NA objectives

Compatible with standard RMS-threaded objectives. Use high-NA plan-apochromat lenses for sensitive detection of dim fluorophores.

🤖 Automated acquisition

Script multi-position, multi-channel, multi-z experiments via Python SDK or the ImSwitch GUI. Tile scanning with autofocus at every position.

📦 Open software

Full control via ImSwitch, Micro-Manager bridge, or REST API. Export to OME-TIFF, OME-Zarr, or any downstream analysis pipeline.
